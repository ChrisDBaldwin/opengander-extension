# Changelog

## 1.0.0

Initial open-source release of the OpenGander Contact Agent browser extension.

- Contact Protocol handshake (propose/accept/decline)
- Ring-based consent model (traffic, interaction, identity)
- Local span storage in IndexedDB
- Session synthesis from raw spans
- Export to JSON file or OTLP/HTTP
- CSV export of session data
- Configurable data retention with automatic cleanup
- Chrome and Firefox support (Manifest V3)
